## Column Guide

| Column | What to write |
|--------|---------------|
| **Name** | Real name or handle (if online-only contact) |
| **Role / Background** | Current role, experience level with systematic strategies |
| **Why Relevant** | Specific reason they match the primary profile — e.g., "builds momentum strategies in Python, posted about abandoning a mean-reversion approach last month" |
| **How Haruki Can Reach** | Direct connection, mutual contact, community member, cold but credible path |
| **Priority** | 1 (strong fit + easy reach), 2 (good fit), 3 (marginal) |
| **Channel** | DM (Twitter/X), Email, LinkedIn, Discord, Slack community, in-person |
| **Outreach Sent?** | Date sent, or blank |
| **Response?** | Yes/No/Pending, date |
| **Interview Scheduled?** | Date/time, or blank |
| **Interview Done?** | Date, or blank |
| **Notes** | Anything relevant — timezone, preferred language, context for personalization |
