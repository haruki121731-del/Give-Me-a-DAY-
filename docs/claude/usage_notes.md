# Usage Notes

## How to use Claude Project
Use Claude Project for:
- product definition
- v1 scope discipline
- internal loop design
- evidence model design
- audit logic
- recommendation logic
- implementation planning
- design review

## How to use Claude Code
Use Claude Code for:
- repo structure
- file creation
- type/schema implementation
- API implementation
- UI implementation
- tests
- refactors

## Connection rule
GitHub is the source of truth.
Claude Project reads and reasons.
Claude Code edits and implements.

## Working rule
Do not ask Claude Project to solve everything at once.
Use one theme per prompt:
- internal loop
- evidence model
- rejection logic
- output package
- UX
- implementation plan

## Warning
If Claude starts sounding like a broad workflow automation consultant, reset the framing immediately.
